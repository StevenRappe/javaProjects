Project: DataSort

Description:
	Creates an ArrayList of 20,000 strings randomly generated using UUID's. Then
	runs a selection sort and a merge sort on the data. Finally it checks
	to see if the two sorts are successful.
	
Files:
	ListSort.java: Contains main method as well as methods to create the
		initial lists, run selection sort, run merge sort, and check to
		see if the lists are sorted.
		
Execution:
	Run file titled "Main". The program is automated and runs independently.
