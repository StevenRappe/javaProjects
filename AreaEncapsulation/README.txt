Project: AreaEncapsulation

Description:
	Creates 100 objects that extend AreaMeasurable randomly chosen
	between the four types. Then calculates the total area
	from the created shapes.
	
Files:
	AreaMeasurable: Interface for all objects.
	Box: Extends AreaMeasurable.
	Circle:	Extends AreaMeasurable.
	Main: Randomly generates objects of the four classes and sums their area.
	Rectangle: Extends AreaMeasurable.
	Sphere: Extends AreaMeasurable.
	
Execution:
	Run program, the program is automated and runs independently.
